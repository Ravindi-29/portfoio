# Personal Portfolio Website

A clean, modern, and beginner-friendly personal portfolio website built from scratch using HTML, CSS, and Vanilla JavaScript.

## Description
This project showcases a complete single-page portfolio layout designed for web developers or designers. It highlights an interactive user interface without relying on external UI libraries or heavy frameworks emphasizing clean code practices, semantic element usage, and mobile responsiveness.

## Features
- **Modern UI/UX Design:** Vibrant color schemes and robust typography.
- **Fully Responsive Layout:** Optimized across mobile phones, tablets, and desktop devices screens (uses CSS Grid and Flexbox).
- **Smooth Scrolling Navigation:** Easy navigation clicking header menu items.
- **JavaScript Form Validation:** Client-side validation ensuring users provide accurate details before form submission.
- **Interactive UI Elements:** Subtle CSS hover and load animations (Progress bars, project cards scaling).

## Technologies Used
- **HTML5:** Semantic architecture.
- **CSS3:** Custom variables, Flexbox, Grid, transitions.
- **Vanilla JavaScript:** DOM Manipulation, event listeners, form validation.
